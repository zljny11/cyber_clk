### 简介
因为嫌弃市面上的番茄钟太无聊，所以我搓了这个***赛博朋克风专注时钟***

界面致敬《赛博朋克 2077》，结合three.js的3D模型，带防摸鱼机制，而且用了 Web Worker 多线程，就算你切到后台去工作，倒计时也不会卡顿!

这款时钟内置精美赛博风壁纸和专属学习用电台，风格各异，支持音量调节，也欢迎你自己上传任何你喜欢的音乐和壁纸！

### 功能说明
- ***硬核防摸鱼***

暂停门槛：想暂停？得先写个 10 个字以上的理由，哪怕是借口也得编像样点!!!

自毁倒计时：暂停超过 10分钟 没回来？任务直接判定失败（神经连接中断）!

- ***复盘系统***

根据专注时长解锁徽章，比如“初出茅厕”、“赛博思想家”等!

自动生成最近十次专注的趋势图，自我监督不断进度吧！

- ***隐私安全***

所有数据（日志、设置、成就）都只存在你自己的浏览器里（localStorage），无需联网，我也看不到!!!!!

- ***自定义***

觉得默认背景不帅？支持上传你自己的壁纸和 BGM!!

### 使用说明

开始入侵：输入你要干啥，点 "开始入侵"!

干活：网页标题会实时显示剩余时间（比如 24:59 | ACTIVE），不用切回来也能看!

结算：完事后给这次专注度打分（1-5星），生成趋势图

看数据：点 DAT 面板看日志和成就

删库跑路：如果想重置所有数据，点 PURGE 按钮清空


### 部署说明

这只是个静态网页，本地部署只需clone然后打开index.html即可
